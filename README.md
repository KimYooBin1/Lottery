# 검지 뽑기 게임

웹캠 앞에 모인 사람들의 검지를 인식해 오늘의 당첨자를 뽑는 브라우저 기반 랜덤 게임입니다.  
버튼을 누르거나 화면을 터치하지 않아도, 참여자는 카메라 앞에 검지를 들어 올리는 것만으로 게임에 참여할 수 있습니다.

## 서비스 소개

검지 뽑기 게임은 회식, 간식 내기, 발표 순서 정하기, 팀 이벤트처럼 가볍고 빠르게 한 명 또는 여러 명을 뽑아야 하는 순간을 위해 만들어졌습니다.

참여자들이 웹캠 앞에 검지를 올리면 서비스가 검지 위치를 인식하고, 화면 안의 참여자가 안정적으로 유지되는 동안 카운트다운을 시작합니다. 중간에 누군가 검지를 빼거나 새로 들어오면 카운트다운은 처음부터 다시 시작됩니다. 모두가 같은 조건에서 준비된 상태일 때만 추첨이 진행됩니다.

## 주요 기능

- 웹캠으로 참여자의 검지를 인식합니다.
- 한 손당 하나의 검지만 참여 대상으로 추적합니다.
- 당첨 인원을 1명 이상으로 설정할 수 있습니다.
- 참여 검지가 추가되거나 사라지면 카운트다운을 자동으로 초기화합니다.
- 당첨된 검지를 결과 사진 위에 노란색 표시와 번호로 보여줍니다.
- 결과 화면에서 당첨 사진을 확인하고 저장할 수 있습니다.
- 지원되는 브라우저에서는 결과 이미지를 바로 공유할 수 있습니다.

## 이용 방법

1. 메인 화면에서 당첨 인원을 선택합니다.
2. `시작하기`를 누르고 브라우저의 카메라 권한을 허용합니다.
3. 참여자들은 웹캠 앞에 검지를 들어 올립니다.
4. 인식된 검지가 안정적으로 유지되면 카운트다운이 시작됩니다.
5. 카운트다운 중 참여자가 바뀌면 자동으로 다시 카운트다운합니다.
6. 추첨이 끝나면 결과 화면에서 당첨 검지와 결과 사진을 확인합니다.
7. 결과 이미지를 저장하거나 공유합니다.

## 이런 상황에 어울립니다

- 누가 커피를 살지 정해야 할 때
- 발표 순서를 가볍게 정하고 싶을 때
- 팀 회의나 워크숍에서 즉석 이벤트를 진행할 때
- 화면 터치 없이 여러 사람이 동시에 참여하는 뽑기 게임이 필요할 때

## 사용 전 확인할 점

- 웹캠이 연결된 데스크톱 브라우저에서 사용하는 것을 권장합니다.
- 브라우저에서 카메라 접근 권한을 허용해야 합니다.
- 검지가 화면 안에 잘 보이도록 손을 너무 겹치지 않게 들어 주세요.
- 조명이 너무 어둡거나 배경이 복잡하면 인식이 불안정할 수 있습니다.
