# Lottery

웹캠으로 손가락을 인식해 당첨자를 뽑는 브라우저 기반 랜덤 게임입니다.

## 주요 기능

- 데스크톱 브라우저 웹캠 기반 참여
- 손가락 단위 랜덤 추첨
- 당첨 인원 수 설정
- 손가락 집합 변화 시 카운트다운 리셋
- 결과 이미지 다운로드
- 지원 브라우저에서 시스템 공유

## 개발

```bash
npm install
npm run dev
```

## 테스트

```bash
npm test
npm run build
```
