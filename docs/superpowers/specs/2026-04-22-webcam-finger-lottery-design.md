# 웹캠 손가락 뽑기 게임 MVP 설계 문서

원본 설계는 로컬 작업 문서에서 정리했고, 구현 기준 요약만 저장한다.

## 핵심 요구사항

- 데스크톱 웹 브라우저에서 동작
- 웹캠 기반 손가락 단위 참여
- 1인당 1~2개 손가락 참여 허용
- 당첨 인원 수 설정 가능
- 손가락 집합 변화 시 카운트다운 리셋
- 결과 이미지를 합성해 저장/공유

## 구현 방향

- React + TypeScript + Vite
- MediaPipe Hands 기반 손 랜드마크 추론
- 손가락 후보 추출 후 활성 객체 추적
- 상태 머신으로 카메라 준비, 안정화, 카운트다운, 결과 상태 관리
- GitHub Actions로 GitHub Pages 배포
